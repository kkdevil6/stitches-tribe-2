## ğŸ’¡ IMAGE TIPS:

### File Formats:
- **Logos**: PNG (with transparent background)
- **Photos**: JPG (smaller file size)
- **Icons**: PNG or SVG

### Sizes:
- **Logo**: 48x48px to 96x96px
- **Service/Process images**: 800x600px (or similar 4:3 ratio)
- **Portfolio images**: Various sizes (masonry layout handles it)
- **Hero logo**: 400-600px width

### Optimization:
- Compress images before adding (use tools like TinyPNG or ImageOptim)
- Aim for file sizes under 200KB for photos, under 50KB for logos
- Keep high quality but optimize for web

### Image Sources:
You can find free stock images at:
- Unsplash.com (free photos)
- Pexels.com (free photos)
- Pixabay.com (free photos)
- Or use your own product photos!
